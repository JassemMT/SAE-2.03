SAE 2.03 : Programmation d’un serveur Web

Introduction

En TP, vous avez programmé un serveur Web capable de :

  • Recevoir des requêtes TCP depuis un navigateur (port 80)
  • Analyser la requête pour trouver un fichier
  • Envoyer ce fichier au navigateur pour affichage

Évolutions prévues

Dans cette SAÉ, le serveur devient configurable et plus puissant :

  • Port d'écoute modifiable
  • Définition d'un répertoire de base
  • Contrôle d'accès aux répertoires
  • Filtrage par adresses IP
  • Journalisation des accès et erreurs
  • Affichage de l'état de la machine
  • Encodage multimédia compatible navigateur
  • Gestion de formulaires HTML
